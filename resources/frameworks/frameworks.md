# Framework Landscape (engineering lens)

## LangGraph
- Strength: graph orchestration + controllability.
- Best for: explicit workflow/state machine systems.

## AutoGen
- Strength: multi-agent conversation + extensibility.
- Best for: research-to-production transition with custom agents.

## LlamaIndex Workflows
- Strength: knowledge-heavy pipelines and data integration.
- Best for: RAG + agent hybrid enterprise use cases.

## CrewAI
- Strength: role/crew abstraction, rapid prototyping.
- Best for: SMB/rapid iteration; must add governance for production.
