# Architecture-Focused Resources
- LangGraph Overview / Repository: https://docs.langchain.com/oss/python/langgraph — graph/state runtime, durable execution, and human-in-the-loop orchestration.
- Microsoft AutoGen: https://github.com/microsoft/autogen — teams, nested teams, memory backends, and engineering-oriented runtime primitives.
- LlamaIndex Multi-Agent Workflows: https://docs.llamaindex.ai/en/stable/understanding/agent/multi_agent/ — orchestrator, AgentWorkflow, and custom planner patterns.
- CrewAI: https://github.com/crewAIInc/crewAI — high-level role/team abstraction, useful as a contrast point to lower-level orchestration stacks.
- MCP launch + spec: https://www.anthropic.com/news/model-context-protocol and https://modelcontextprotocol.io/ — protocolization affects architecture because tools and data become part of the runtime surface.
- Key failure lens: pair architecture reading with `Why Do Multi-Agent LLM Systems Fail?` for drift, loop, mismatch, and verification-collapse categories.
