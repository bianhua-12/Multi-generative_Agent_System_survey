# Memory / State-Focused Resources

- LangGraph Overview / Repository: https://docs.langchain.com/oss/python/langgraph — explicit state transitions, durable execution, checkpoints, and replay-oriented design.
- Microsoft AutoGen: https://github.com/microsoft/autogen — release line signals practical movement from chat-style coordination to teams plus memory backends.
- LlamaIndex Multi-Agent Workflows: https://docs.llamaindex.ai/en/stable/understanding/agent/multi_agent/ — useful for understanding how retrieval, workflow state, and agent handoffs interact.
- Model Context Protocol launch + spec: https://www.anthropic.com/news/model-context-protocol and https://modelcontextprotocol.io/ — standardized external context/tool access changes how "memory" is distributed across systems.
- OpenAI Platform Documentation (Agents / Responses / tracing): https://platform.openai.com/docs — tracing and tool/runtime semantics matter because memory without observability is not operable state.
