# Industry Signals (2025–2026Q1)

> Note: This page keeps Tier A and strong Tier B evidence only. Uncertain claims are explicitly marked.

- OpenAI: stronger emphasis on runtime systems (tools/handoffs/tracing/evals), beyond single-turn prompting.
- Anthropic: MCP ecosystem pushes model-tool-data standardization while surfacing protocol-security governance concerns.
- Google/Cloud: interoperability and enterprise workflow integration become recurring platform narratives.
- Microsoft/GitHub: trajectory from AutoGen-style experimentation toward developer-workflow-embedded agents.
- LangChain/LangGraph/LlamaIndex/CrewAI: ecosystem moves from demos to observability/testing/deployability.

## Uncertain Items (continuous verification needed)
- Whether A2A-like efforts converge to durable cross-vendor semantic interoperability.
- Whether MCP server security baselines standardize during 2026.
